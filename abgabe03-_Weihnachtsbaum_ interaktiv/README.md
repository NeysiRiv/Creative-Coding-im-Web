# Abgabe 3 – Weihnachtsbaum mit interaktiven Lichtern

In diesem Projekt wurde eine kleine interaktive Weihnachtsseite mit HTML, CSS und JavaScript erstellt.
Ein stilisierter Weihnachtsbaum wird angezeigt, dessen Lichter per Button ein- und ausgeschaltet werden können.
Zusätzlich können die Farben der Lichter geändert werden.

## Optional wurden Animationen wie Blinken oder Schneefall umgesetzt.

## HTML-Struktur

Überschrift mit einem weihnachtlichen Gruß (z. B. Fröhliche Weihnachten!)

Button zum Ein- und Ausschalten der Lichter

Dropdown-Menü zur Auswahl der Lichtfarbe (Gelb, Blau, Rot)

Baum und Lichter über semantisch sinnvolle Elemente aufgebaut

## CSS-Gestaltung

Weihnachtsbaum als dreieckige Form über border erstellt

Kleine Kreise als Lichter auf dem Baum positioniert

Sanfte Ein- und Ausblendanimation mit transition

Farbwechsel der Lichter per CSS-Klassen gesteuert

## JavaScript-Funktionen

Funktion zum Ein- und Ausschalten aller Lichter

Farbänderung basierend auf der Auswahl im Dropdown

Nutzung von:

3 Konstanten

2 forEach-Schleifen zur Änderung aller Licht-Elemente

2 Event Listenern:

Klick auf den Button

Änderung der Farbauswahl

## Optionale Erweiterungen (Extra-Punkte)

Blinkende Lichter mit setInterval()

Schneefall-Animation über CSS oder JavaScript

Verwendete Technologien

HTML5

CSS3 (Transition, Positionierung, Formen)

JavaScript (DOM-Manipulation, Events, Klassenwechsel)

# FH Oberösterreich – Design of Digital Products (DDP)
# Wintersemester 2024
# Creative-Coding-im-Web

Dieses Repository enthält alle vier Projekte aus dem Kurs Creativ Coding im Web. Die Aufgaben decken grundlegende Webentwicklungsthemen ab: HTML, CSS, Flexbox, Grid, JavaScript, DOM-Manipulation, Event Listener, Animationen und interaktive Elemente.

Jedes Projekt befindet sich in einem eigenen Ordner mit einer eigenen Projektbeschreibung (README).

Inhalte des Repositories
## Abgabe 1 – Webseite für eine Schokoladenfabrik

Ordner: abgabe01-schokoladenfabrik
HTML- und CSS-Grundprojekt mit Fokus auf semantischer Struktur, Navigation, Bildern, Typografie, Listen, Pseudoklassen und Flexbox.

Details: im README der Abgabe.

## Abgabe 2 – Event-Portal „Casting für den Musikvideo-Star“

Ordner: abgabe02-musikvideo-casting
Erweiterung einer Event-Webseite inklusive Tabellen, Listen, Grid-Layout, Flexbox, Hintergrundbildern und Multimedia-Elementen.

Details: im README der Abgabe.

## Abgabe 3 – Weihnachtsbaum mit interaktiven Lichtern

Ordner: abgabe03-weihnachtsbaum
Interaktive Weihnachtsseite mit HTML, CSS und JavaScript: Lichter ein-/ausschalten, Farben ändern, Animationen. Optional: blinkende Lichter und Schneefall.

Details: im README der Abgabe.

## Abgabe 4 – Interaktives Quiz

Ordner: abgabe04-interaktives-quiz
Dynamisches Quiz mit JavaScript: Fragen aus Arrays generieren, Benutzerantworten prüfen, Punktzahl ausgeben, optional Fehler anzeigen.

Details: im README der Abgabe.

Verwendete Technologien

HTML5 (Struktur, Semantik, Multimedia)

CSS3 (Flexbox, Grid, Animationen, Typografie)

JavaScript (DOM-Manipulation, Events, Arrays)

Google Fonts

Teilweise Responsive Design
